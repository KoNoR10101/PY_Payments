Дорогой пользователь, перед использованием прочти эту инструкцию.

1. Для работы нам понадобится Python, версии 3.10 или выше
2. Чтобы увидеть код нам понадобится IDE, советую скачать PyCharm.
3. После установки необходимо скачать недостоющии библиотеки,
    они находятся в файле requirements
4. вам понадобится токен, его можно получить у botfather в телеграмме.
5. готово, можете пользоваться


Проект является open source поэтому оцените его на github https://github.com/KoNoR10101/PY_Payments
